# DualRef-TTS Demo

Demo page for *DualRef-TTS: A Label-Free Dual-Reference Paradigm for Independent Style-Timbre Control in Zero-Shot Speech Synthesis*.

Open `index.html` locally or deploy via GitHub Pages.

## Layout
- `index.html` — demo page
- `samples/` — wav files referenced by the page
  - `part<N>/<idx>/source_timbre.wav` — timbre reference
  - `part<N>/<idx>/source_emotion.wav` — style reference
  - `part<N>/<idx>/generated.wav` — synthesized output
